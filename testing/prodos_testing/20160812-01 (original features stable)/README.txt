First stable version that support the read byte length and seek. But, it has bugs with the seek feature. 

Read, write, seek and get file length all tested. 


Tested successfully on the following platform, in unenhanced IIe mode


AppleWIN harddive
CFFA Harddrive
AppleWIN floppy (disk 1 & 2)
Virtual II floppy (disk 1 & 2)
Real floppy (disk 1 & 2)
CFFA Floppy (disk 1 & 2)
